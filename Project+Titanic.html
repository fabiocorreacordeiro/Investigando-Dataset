por Fbio Corra Cordeiro

Este projeto tem como objetivo analisar a base de dados titanic_data.csv que contm informaes de 891 passageiros do Titanic. A base foi obtida no site Kaggle. Neste trabalho queremos avaliar se o princpio de que mulheres e crianas tm preferncias nos salvamentos de um naufrgio foi respeitado no caso do famoso naufrgio do Titanic.

Pergunta:¶

A pergunta que ser respondida nessa anlise se a prtica de salvar mulheres e crianas primeiro foi respeitada no naufrgio do Titanic. Para tal iremos verificar se a quantidade de mulheres e crianas que sobreviverem ao naufrgio estatisticamente mais maior do que de homens.

Para isso vamos criar uma funo que separa qualquer uma das colunas da tabela "titanic_data_age" em duas e agrupa os dados pela idade. Essa funo ir nos ajudar a montar os grficos

def plt_graph1(feature, parameter):
    
    """"
    For a column of titanic_data_age the plt_graph1 do:
    - split the column in feature == parameter and feature == parameter;
    - then group the both data by 'Age'
    Return the column splited
    """
    
    feature_parameter = titanic_data_age[titanic_data_age[feature] == parameter]
    feature_parameter = feature_parameter.groupby(['Age']).size()
    feature_not_parameter = titanic_data_age[titanic_data_age[feature] != parameter]
    feature_not_parameter = feature_not_parameter.groupby(['Age']).size()
    return(feature_parameter, feature_not_parameter)

def plt_graph2(data, feature):
    """"
    For one of two data ("titanic_data_sex" or "titanic_data_age"), 
    the dataframe is grouped by column "feature" and by "Survived".
    plt_graph2 return bar_1 and bar_2:
    bar_1 = feature[0] not survived and feature[1] not survived
    bar_2 = feature[0] survived and feature[1] survived
    """
    feature_survived = data.groupby([feature,'Survived']).size()
    feature_p0_notsurvived = feature_survived[0]
    feature_p0_survived= feature_survived[1]
    feature_p1_notsurvived= feature_survived[2]
    feature_p1_survived= feature_survived[3]
    bar_1 = [feature_p0_notsurvived,feature_p1_notsurvived]
    bar_2 = [feature_p0_survived,feature_p1_survived]
    return bar_1, bar_2

Agora vamos fazer os testes de inferncia estatstica para verificar se a diferena da proporo de sobreviventes e mortos so significativas. Usaremos a distribuio t de student para compararmos as duas amostras.

Primeiramente iremos anunciar a Hiptese nula (Ho) sendo que a proporo de homens sobreviventes igual a proporo de mulheres sobreviventes. E da mesma maneira que proporo de adultos sobreviventes igual a proporo de crianas sobreviventes.

Conclusão¶

Para confirmarmos as hipteses nulas (Ho) de que a proporo de sobreviventes foram iguais deveramos encontrar valores de p acima de 0,05. Como os valores foram bem abaixo rejeitamos as hipteses nula. Portanto podemos constatar, com base na amostra analisada, que a morte de mulheres e crianas foram estatiticamente inferior de homens e adultos no naufrgio do Titanic.

No entanto, como est faltando informaes de aproximadamente 400 passageiros, ainda ficamos com algumas questes em abertos. Caso haja algum vis nos dados ausente (todos sejam de mulheres ou criancas que no sobreviveram, por exemplo) poderamos chegar em concluses diverentes da anlise aqui realizada.

Uma continuidade possvel para essa anlise verificar se mulheres e crianas tiveram a mesma chance de sobreviver considerando o quanto elas pagaram pela passagem. Ou mesmo se mulheres e crianas que pagaram tarifas baixas tiveram mais chance de sobreviver do que homens que pagaram tarifas mais caras.
